1) Extract all the files from the compressed ZIP
2) First open the EXE file this is to install Python, if you already have Python 3.12.2 or above then skip to step two.
    -When installing Python, ensure the button to add python to PATH is checked.
    -Then Click customise installation
    -Then make sure that all buttons are checked
    -Click next
    -Tick the first five boxes but nothing else, ignore the path below
    -Click install then Yes
    -Finally click the 'Disable Path Length Limit' button, yes, then close
3) Second, open up the setup file (.cmd). This installs the required modules if needed. Run this whether or not you think they are installed.
4) Make sure to read (and edit) the settings file (.py)
5) Then you should be ready to go. Open up the main.py
